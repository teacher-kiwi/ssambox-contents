## 우주의 크기 — Pixel Space

달의 지름을 1픽셀로 축소했을 때, 태양계의 실제 크기를 체험할 수 있는 인터랙티브 시뮬레이션입니다.

### 사용 방법

- **마우스 스크롤 / 드래그**: 좌우로 이동
- **화살표 키(←→)**: 좌우로 이동
- **스페이스바**: 광속 자동 이동 시작/정지
- **하단 속도 버튼**: 정지 / 광속(1×) / 100× / 1000× / 10000× / 워프
- **상단 미니맵**: 클릭하여 원하는 위치로 즉시 이동
- **전체화면 버튼**: 전체화면 전환

### 관찰 포인트

🌕 달의 지름이 고작 1픽셀인데, 태양은 얼마나 클까요?
🌍 지구에서 화성까지 얼마나 먼지 스크롤로 느껴보세요.
🪐 목성과 토성 사이의 광활한 빈 공간을 체험해 보세요.
🚀 빛의 속도로 달려도 명왕성까지 얼마나 걸릴까요?

### 학습 목표

- 태양계 천체들의 실제 크기 비율을 이해한다.
- 태양계의 광대한 거리감을 체험적으로 느낀다.
- 빛의 속도와 우주적 거리를 비교하여 우주의 스케일을 인식한다.
