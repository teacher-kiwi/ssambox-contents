## 구조

```
_shared/          → 공통 리소스 (CSS/JS)
_template/        → 새 콘텐츠 생성 시 복사할 템플릿
solar-system/     → 태양계 시뮬레이션 (p5.js)
```

## 새 콘텐츠 만들기

1. `_template/` 폴더를 복사하여 새 폴더 생성 (영문 소문자, 하이픈 구분)
2. `readme.md`의 프론트매터 수정 (title, grade, subject, area, tags)
3. `index.html`에 콘텐츠 코드 작성
4. `thumbnail.png` 추가 (권장)
5. 이 README의 콘텐츠 목록에 추가
6. GitHub에 push → 서버가 자동으로 동기화

## 프론트매터 예시

```yaml
---
title: 태양계 시뮬레이션
grade: 5
subject: 과학
area: 지구와 우주
tags: [태양계, 행성, 천문, 우주]
content_type: simulation
---
```

## 기술 스택

- **p5.js** — 2D 시뮬레이션, 수학/과학 시각화
- **three.js** — 3D 시뮬레이션, 입체 모델
- **SsamBox 공통 프레임워크** — 일관된 UI (컨트롤바, 전체화면 등)
