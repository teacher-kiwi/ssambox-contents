## 태양계 시뮬레이션

태양과 8개의 행성이 공전하는 모습을 실시간으로 관찰할 수 있는 인터랙티브 시뮬레이션입니다.

### 사용 방법

- **마우스 드래그**: 태양계 이동
- **스크롤**: 확대 / 축소
- **행성에 마우스를 올려보세요**: 행성 정보 표시
- **전체화면 버튼**: 전체화면 전환

### 관찰 포인트

🔭 행성마다 공전 속도가 다른 이유는 무엇일까요?
🌍 지구 옆에 작은 점은 무엇일까요?
🪐 토성의 고리는 무엇으로 이루어져 있을까요?

### 학습 목표

- 태양계를 구성하는 천체의 종류와 특징을 이해한다.
- 내행성과 외행성의 차이를 비교할 수 있다.
- 행성의 공전 주기와 궤도 크기의 관계를 탐구한다.
